When a user places an order on this website, he/she receives an order confirmation page that contains all the important information about the order.\\\\
On the left side of the page is the order number and an acknowledgement of the order. Below that, it says that the order has been confirmed and that the user will receive more information about the delivery shortly.\\\\
On the right side of the page is an order summary that lists all items purchased, the number of products ordered, and the total price of the order.\\\\
Below the thank you note, customer information is displayed, including the user's shipping and billing address and the shipping and payment method selected.\\\\
The order confirmation page provides the user with a quick and easy way to check the details of their order and make sure everything is correct.
