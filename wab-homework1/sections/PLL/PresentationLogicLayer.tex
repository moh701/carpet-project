OCS contains various pages that deal with the sale and purchase of various carpet tables. In the following we define the pages that we will develop for our online carpet store. All pages will be developed with jsp..

\begin{itemize}
    \item Home page: The home page is also the homepage. This is the first page that the user can see.
    \item Registration page: This page allows the user to register.
    \item Login page: This page allows the user to log in to their existing account.
    \item Forgot Password Page: If a user forgets his password he gets the possibility to set a new password here.
     \item Product Page: This page contains all information about a product.
    \item Messages: Here the buyer can write with the seller about a product, ask questions and submit offers.
    \item Shopping Cart: This page gives an overview of the products that are in the shopping cart.
     \item Checkout - Customer Info: On this page the buyer enters his personal data.
    \item Checkout - Shipping: Here the customer decides which delivery options are best for him. 
    \item Checkout - Payment: On this page the payment options are checked.
    \item Checkout - Success: This is an order confirmation on the page.
    \item Email - Confirmation: After receiving the order confirmation, the order process is completed.
\end{itemize}
